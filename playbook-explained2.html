‚ùì Question: What does the following playbook do?

- name: Install common tools
  hosts: all
  tasks:
    - name: Install packages
      apt:
        name: "{{ item }}"
        state: present
      loop:
        - git
        - curl
        - rsync

    - name: Only on Ubuntu
      debug:
        msg: "This is Ubuntu"
      when: ansible_facts['os_family'] == "Debian"

üìò Playbook Overview

  • name: Describes the play's purpose: installing common tools.
  • hosts: all ‚Äî Targets all hosts in the inventory.
  • tasks: Contains the actions to be executed.

üîÅ Task Breakdown

1. Install Packages

  • Uses the apt module to install multiple packages.
  • loop: iterates over a list: git, curl, and rsync.
  • name: "{{ item }}" dynamically inserts each package name using Jinja2 templating.
  • state: present ensures each package is installed.
üí° How the loop works:
The loop: provides a list of items: git, curl, and rsync.
For each item in the list, Ansible replaces {{ item }} with the current value.
So the task runs three times:
  • First with name: git
  • Then with name: curl
  • Finally with name: rsync
This makes the task reusable and efficient.

2. Conditional Debug Message

  • Uses the debug module to print a message.
  • Runs only if the host's OS family is Debian (which includes Ubuntu).
  • Uses ansible_facts['os_family'] to determine the OS type.
