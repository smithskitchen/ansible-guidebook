- name: Configure web server
  hosts: web
  become: yes
  tasks:
    - name: Install nginx
      apt:
        name: nginx
        state: present

    - name: Deploy index.html
      copy:
        src: ./index.html
        dest: /var/www/html/index.html
      notify: Reload nginx

  handlers:
    - name: Reload nginx
      service:
        name: nginx
        state: reloaded

üìò Playbook Overview

  • name: Describes the play's purpose.
  • hosts: web ‚Äî Targets servers in the "web" group.
  • become: yes ‚Äî Uses elevated privileges (sudo).

üõ† Tasks Breakdown

1. Install nginx

  • Uses the apt module to install nginx.
  • state: present ensures it's installed.

2. Deploy index.html

  • Uses the copy module to place a local HTML file on the server.
  • If the file changes, it triggers the Reload nginx handler.

üîÅ Handlers

  • Reload nginx ‚Äî Reloads the nginx service only if notified.
  • Efficient: avoids unnecessary reloads unless changes occur.
